A modern, dark-themed wallpaper gallery featuring over 3,800+ high-quality / 4K wallpapers organized into categories.

![og](https://github.com/user-attachments/assets/572adcc7-f9cc-410d-8a62-8125f93bafae)



## Features

- Dark glassmorphism UI
- Category filtering with smooth animations
- Search functionality
- Lazy loading for performance
- Responsive design (desktop, tablet, mobile)
- 4K wallpaper support
- Direct download and open options

## Categories

- Abstract, Anime, Architecture, Art
- Digital, Gaming, General
- Minimal, Nature
- Other, Space
- 4K Ultra HD

## Usage

Browse wallpapers by category using the bottom navigation bar, or search for specific themes using the search bar.

[Open Wallpaper Archive](https://wallpaper-archive.pages.dev/)

## Wallpaper Sources

Wallpapers are curated from various open-source collections:

- [walls](https://github.com/dharmx/walls)
- [aesthetic wallpapers](https://github.com/D3Ext/aesthetic-wallpapers/) - Additional wallpapers
- [onedark](https://github.com/Narmis-E/onedark-wallpapers) - OneDark themed
- [Wallpaper-Ban](https://github.com/JaKooLit/Wallpaper-Bank) - Curated bank
- [NASA](https://www.nasa.gov/gallery/nasa-spacewalkers/) - Space & astronomy
- [London](https://www.nationalgallery.org.uk/) - Digital art & illustrations
- [lwalpapers](https://github.com/whoisYoges/lwalpapers#) - General collection
